Deep Learning for Experimental Psychologists and Cognitive Neuroscientists#

This Jupyter Book covers several deep-learning topics building slowly from basic concepts to more advanced usages. The materials in this book initiated originally from the teaching syllabus for the deep learning course taught at Brain and Behaviour Master programme taught at Justus-Liebig University of Giessen. While some examples are tailored for Experimental Psychologists and Cognitive Neuroscientists, the content should be useful to anyone interested in deep learning.
